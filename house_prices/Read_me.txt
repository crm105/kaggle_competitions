Description: This document contains a Read_me description for all files found in the ames_housing project folder. Models tested in this project include a Gradient Boosting Machine, Ridge Regression, Random Forest, and K-Nearest Neighbors


"ames_eda.R" - This script contains the main workflow for the housing price prediction model featured in this project folder. Data cleaning, model fitting, and ensembling tasks are all featured in this script.

Input Files:  "train.csv" ; "test.csv"

Output Files: "predictions.csv"
